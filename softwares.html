Run section-by-section ( %%) without debugger:

  • ⌘+Enter: Run current section of .m file.
  • ⌘+↓: Switch sections.

Run section with debugger (code will stop at each breakpoint)

  • ⌘+ALT+R: Run .m file to next breakpoint
  • ⌘+ALT+R: Resume running to next breakpoint
  • MAJ+fn+f5: Exit debugging mode
